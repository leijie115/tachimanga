# Tachimanga Extension Repository

Custom extension repository for [Tachimanga](https://tachimanga.app/).

## Available Sources

| Source | Language | URL |
|--------|----------|-----|
| Manga18fx | English | https://manga18fx.com |
| 漫小肆韓漫 (Ikanwzd) | 中文 | https://www.ikanwzd.cc |
| 韩漫库 (Se8) | 中文 | https://se8.us |
| 18漫畫 (EighteenMH) | 中文 | https://18mh.org |
| 巴卡漫画 (Bakamh) | 中文 | https://bakamh.com |

> **Note:** All sources contain NSFW (18+) content.

## How to Use

### 1. Add Repository

Open Tachimanga, go to **More → Extensions → Extension Repositories → Add Repository**, and paste the following URL:

```
https://raw.githubusercontent.com/leijie115/tachimanga/main/repo/index.min.json
```

### 2. Install Extensions

Go to **Browse → Extensions**, find the extension you want in the list, and tap **Install**.

### 3. Browse Manga

After installation, go to **Browse → Sources**, select the source you installed, and start reading.

# Tachimanga 扩展仓库

适用于 [Tachimanga](https://tachimanga.app/) 的自定义扩展仓库。

## 可用图源

| 图源 | 语言 | 网址 |
|------|------|------|
| Manga18fx | English | https://manga18fx.com |
| 漫小肆韓漫 (Ikanwzd) | 中文 | https://www.ikanwzd.cc |
| 韩漫库 (Se8) | 中文 | https://se8.us |
| 18漫畫 (EighteenMH) | 中文 | https://18mh.org |
| 巴卡漫画 (Bakamh) | 中文 | https://bakamh.com |

> **注意：** 以上图源均包含 NSFW（18+）内容。

## 使用方法

### 1. 添加仓库

打开 Tachimanga，进入 **更多 → 扩展 → 扩展仓库 → 添加仓库**，粘贴以下地址：

```
https://raw.githubusercontent.com/leijie115/tachimanga/main/repo/index.min.json
```

### 2. 安装扩展

进入 **浏览 → 插件**，找到想要的扩展，点击 **安装**。

### 3. 开始阅读

安装完成后，进入 **浏览 → 图源**，选择已安装的图源即可开始阅读。
